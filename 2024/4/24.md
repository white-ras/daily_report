- 画像ファイルは / とすると、publicを指定する（省略できる）
- メタデータ　→画面にはでてこない裏データ

- nfe Next function export
- css を分ける　`styles`と`module` 汎用性がある場合はstyles。骨格。
- layout.module.cssはlayout限定なので汎用性がない
- globals.css
- 動的ルーティング `[id].js`
- get static path
